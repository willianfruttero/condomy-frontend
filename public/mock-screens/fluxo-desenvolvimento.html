3. Fluxograma Passo a Passo

Passo 1: Protótipos HTML

Crie arquivos semânticos (header/main/footer), CSS mínimo e comentários.

↓

Passo 2: Revisão UI/UX

Ajuste cores, espaçamentos, responsividade básica e acessibilidade (WCAG 2.1 AA).

↓

Passo 3: React + TypeScript

Transforme HTML em JSX, adicione props tipadas, estilos e testes unitários.

↓

Passo 4: QA & Performance

Jest/RTL, Cypress E2E, Sonar quality gate, Lighthouse ≥ 90, bundle analysis.

↓

Passo 5: Deploy & Observabilidade

CI/CD (GitHub Actions), Kubernetes (Rolling Update), ELK, Prometheus/Grafana, alertas.

4. Mapa Detalhado de Conexões

Lógica

Hooks, context, state, validações Zod, chamadas Axios/GraphQL.

UI/UX

Layouts, responsividade, acessibilidade, fluxos e protótipos.

Estilo

Design tokens, Tailwind config, Styled-Components, animações.

Dados

Cache React Query, normalização, revalidação, interceptors.

Infra

Terraform, Helm charts, CI/CD, monitoramento, segurança e IAM.
